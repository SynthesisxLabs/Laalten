## ⚡ Install

Open **Terminal** and paste:

```bash
curl -sL https://raw.githubusercontent.com/SynthesisxLabs/Laalten/main/install.sh | bash
```

That's it. Laalten will download, install to `/Applications`, and launch automatically.

### Manual Install

1. Download the latest `.dmg` from [**Releases**](https://github.com/SynthesisxLabs/Laalten/releases/latest)
2. Open the DMG and drag **Laalten.app** to **Applications**
3. Launch from Spotlight or Applications folder

## ✦ Features

| Feature | Description |
|---------|-------------|
| 🎵 **Music Controls** | Now Playing widget with album art, playback controls, and live waveform visualizer |
| 📋 **Clipboard Manager** | Instant access to clipboard history with one-click copy |
| 📁 **File Shelf** | Drag files onto the notch to stash them — drag out to any app or AirDrop |
| 📅 **Calendar** | Real-time EventKit integration showing today's schedule |
| 🔋 **Battery & System** | Live battery percentage, CPU usage, and system stats |
| 📝 **Notepad** | Quick notes with Import to Apple Notes |
| 🪞 **Mirror** | Built-in camera preview widget |
| ⌨️ **Typing Master** | Fun typing speed game right in your notch |
| 🎯 **HUD Overlays** | Premium volume/brightness indicators |
| ✨ **Hello Animation** | Beautiful startup sequence |

## 🖥️ Requirements

- **macOS 14 (Sonoma)** or later
- MacBook with Apple Silicon (M1/M2/M3/M4) or Intel
- Works with or without a physical notch

---

## 🔄 Update

Re-run the install command to update to the latest version:

```bash
curl -sL https://raw.githubusercontent.com/SynthesisxLabs/Laalten/main/install.sh | bash
```

---

## 🗑️ Uninstall

```bash
rm -rf /Applications/Laalten.app
```
